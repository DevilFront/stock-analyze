## 서비스 컨셉: AI 기업 공시·실적 리포트

- **핵심 한 줄**:  
  내 관심 종목의 공시·실적·뉴스를 자동으로 모으고,  
  한 장짜리 AI 리포트 + 핵심 지표 대시보드로 요약해주는 서비스.

- **대상 사용자**:  
  - 개별 종목을 공부하긴 해야겠는데, 공시/실적/뉴스를 일일이 읽기 벅찬 개인 투자자  
  - 바쁜 직장인, 장기/퀀트 투자자, 리포트 요약만 빨리 보고 싶은 사람

---

## 주요 기능

1. **기업 데이터 수집 (백엔드)**
   - `/company/raw?symbol=` FastAPI 엔드포인트에서:
     - 회사 프로필 (업종, 시총, 밸류에이션 지표 등)
     - 최근 4분기/3년 재무 데이터 (매출, 영업이익, 순이익, EPS 등)
     - 최근 공시 목록 (제목, 날짜, 링크, 간단 요약)
     - 주요 뉴스 헤드라인/요약
   - 초기 단계에서는 삼성전자 등 몇 개 종목을 Mock JSON으로 구현 후,
     이후 실제 공시/실적/뉴스 API로 교체.

2. **AI 기업 리포트 생성 (Vercel AI / OpenAI)**
   - 입력:
     - 기업 프로필 + 실적 추이 + 공시/뉴스 요약 + 리스크 지표
   - 출력(마크다운 구조):
     - `1) 분기 실적 요약`
     - `2) 성장성/수익성/재무 건전성 평가`
     - `3) 최근 공시·뉴스의 핵심 포인트`
     - `4) 리스크 및 체크 포인트`
   - 프롬프트 원칙:
     - "데이터 부족/판단 어려움" 금지, 대신 전문 용어로 해석
     - 모든 문장에 숫자 최대한 포함
     - 상방/하방 시나리오를 분리해서 제시
     - 투자 권유 금지, 데이터 기반 특징 설명에 집중

3. **리포트 UI (Next.js + Tailwind + shadcn)**
   - 라우트: `/report/[symbol]`
   - 상단 요약:
     - 기업명/티커/업종/시총
     - 최근 실적 한 줄 요약 (매출/이익 증감, 아이콘 포함)
     - 요약 카드 2~3개 (성장성, 수익성, 리스크 요약)
   - 본문:
     - 좌측: 재무/실적 차트 (매출/영업이익 라인 차트, EPS/ROE 바 차트 등)
     - 우측: AI 리포트 카드 (마크다운 렌더링, 스트리밍 타이핑)

4. **대시보드 (홈)**
   - 메인 검색:
     - 종목 코드/회사명 입력 → "기업 리포트 보기"
   - 사이드바:
     - `최근 본 리포트`, `관심 기업`, `마이페이지`
   - 이후 확장:
     - 관심 기업에 공시/실적 업데이트 발생 시 알림 + 자동 리포트 생성

## 향후 확장 아이디어

- **알림/구독 모델**
  - 관심 종목에 새 공시/실적/뉴스가 나오면 → 이메일/웹푸시 + 자동 리포트 링크 발송.
  - 월 구독제로 리포트 횟수/알림 개수 제한.

- **리포트 히스토리**
  - 각 종목의 과거 리포트를 저장하고,
  - 실적 발표/이벤트 전후 리포트를 비교하는 기능.

- **전략/포트폴리오 연동**
  - 사용자가 보유/관심 포트폴리오를 등록하면,
  - 분기마다 "포트폴리오 전체 실적 리포트"를 생성하는 기능.
